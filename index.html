Solo project · NFC / Mobile

TrackIt! (name may change)

A database-driven mobile app for tracking physical items using NFC stickers. Users register any item they want to monitor, program an NFC sticker to it, then tap their phone to instantly toggle that item between in and away — no manual searching, no spreadsheets. Built to work for any environment: home, job site, warehouse, or kit room. The backend is structured around a relational database that maps each NFC tag ID to a user's item records, with real-time state updates on every tap.
